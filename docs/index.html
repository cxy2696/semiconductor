行业基础知识

来源：Wikipedia API
1. 半导体，是一种电导率在绝缘体至导体之间的物质或材料。
2. 在一定温度范围内，半导体的电荷载流子浓度会随着温度升高而增加，在宏观上表现为电导率上升、电阻率下降；而在绝对零度时，则成为绝缘体。
3. 依有无加入掺杂剂，半导体可分为：本征半导体、杂质半导体（n型半导体、p型半导体）。

术语速览

上游（设备/材料）
定位：提供光刻、刻蚀、薄膜沉积、硅片与化学品等基础能力。案例公司：北方华创、中微公司、沪硅产业。典型行业：晶圆制造、先进封装产线建设。
中游（设计/制造）
定位：完成芯片架构设计、流片与量产制造，是技术和资本密集环节。案例公司：寒武纪、兆易创新、中芯国际。典型行业：AI算力、消费电子、工业控制。
下游（封测/应用）
定位：负责封装测试并将芯片导入终端场景，直接影响交付质量与可靠性。案例公司：长电科技、通富微电、华天科技。典型行业：汽车电子、通信设备、智能终端。
Fabless
专注芯片设计，不自建晶圆厂，依赖外部代工生产。
Foundry(晶圆代工)
提供制造能力，承接Fabless和部分IDM的量产需求。
IDM
覆盖设计、制造、封测的一体化模式，资本开支较高。
封测
对芯片进行封装与测试，直接影响良率和交付质量。
成熟制程/先进制程
成熟制程偏工业与车规，先进制程偏高性能计算与AI。
国产替代
关键设备、材料和芯片能力本土化，是中长期主线。
